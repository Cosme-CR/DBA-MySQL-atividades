3. Defina o que é um SGBD, e qual a importância que um SGBD tem em
relação a um software de banco de dados.

SGBD (Sistema de Gerenciamento de Banco de Dados) é o software que atua 
como uma camada intermediária entre o usuário (ou aplicação) e o 
banco de dados físico. que controla a criação, armazenamento, acesso, 
administração e segurança dos bancos de dados,



4. Descreva qual a função dos métodos DDL, DML e DCL em uma estrutura de
um SGBD.

DDL (Data Definition Language): comandos para definir/alterar a estrutura do banco — CREATE, ALTER, DROP, TRUNCATE (tabelas, índices, esquemas).
DML (Data Manipulation Language): comandos para manipular os dados contidos nas estruturas — SELECT, INSERT, UPDATE, DELETE.
DCL (Data Control Language): comandos para controlar acesso e permissões — GRANT, REVOKE (controle de privilégios)
